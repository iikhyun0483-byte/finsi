**데이터 소스**:
- Yahoo Finance (주식)
- Binance API (암호화폐)
- 실시간 매크로 지표

### 5. `/backtest` - 백테스팅 & 신호 검증 📈
**기능**:

#### A. 신호 검증 탭 (새로 추가!)
- **과거 2년 데이터**로 신호 승률 검증
- 1주일/1개월/3개월 승률 계산
- 평균 수익률 계산
- 발생 신호 목록 표시
- 차트 시각화

**테스트 결과** (실제 데이터):
```
SPY @ threshold=55:
  - 신호 수: 125개
  - 1주일 승률: 66%
  - 1개월 승률: 85%
  - 3개월 승률: 94%
  - 평균 수익률: 1.65% (1개월)

BTC @ threshold=60:
  - 신호 수: 154개
  - 승률: ~50% (동전 던지기)
```

**중요**:
- 중립 매크로 지표 사용 (백테스트 환경)
- 최고 점수 ~68점 (실시간보다 낮음)
- 권장 threshold: 50-65점

#### B. 전략 백테스트 탭
- 6가지 투자 전략 지원
  1. Buy & Hold (장기보유)
  2. Golden/Dead Cross (이동평균선)
  3. RSI Reversal (과매도/과매수)
  4. MACD Crossover
  5. Bollinger Band (밴드 반등)
  6. Dual Momentum (12개월 모멘텀)

**통계**:
- CAGR (연평균 수익률)
- MDD (최대 낙폭)
- 샤프 비율
- 승률
- 최종 자산

**기능**:
- 모의 자동매매 시스템 (실제 거래 X)
- 5분마다 신호 체크 (백그라운드)
- 브라우저 알림 (Notification API)
- 거래 로그 저장 (localStorage)
- 전략 설정
  - 신호 기준점 (50-90)
  - 투자금액
  - 손절률 (-10% ~ 0%)
  - 익절률 (5% ~ 50%)

**작동 방식**:
1. 전략 활성화
2. 5분마다 신호 점수 확인
3. 기준점 이상 → 매수 알림
4. 보유 중 → 손절/익절 체크
5. 거래 로그 기록

**주의**:
- 실제 거래 아님 (모의)
- 브라우저 열려있어야 작동
- localStorage에 기록 저장

## 📊 백테스팅 시스템

### 신호 검증 방식
```typescript
1. 과거 2년 데이터 수집 (500+ 거래일)
2. 각 시점마다:
   - 과거 300일 가격 데이터로 신호 생성
   - 점수가 threshold 이상이면 기록
3. 각 신호마다:
   - 1주일 후 (5 거래일) 수익률 계산
   - 1개월 후 (21 거래일) 수익률 계산
   - 3개월 후 (63 거래일) 수익률 계산
4. 통계:
   - 승률 = (수익 난 신호 / 전체 신호) × 100
   - 평균 수익률 = Σ수익률 / 신호 수
```

### 중요 사항
- **중립 매크로 사용**: fearGreed=50, vix=15, fedRate=2.5, buffett=100
- **이유**: 과거 시점의 실제 매크로 데이터 없음
- **결과**: 백테스트 점수가 실시간보다 낮음 (최고 ~68점)
- **권장 threshold**: 50-65점
